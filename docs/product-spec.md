# VocabMaster 产品规格

## 2.0 完整版目标

VocabMaster 2.0 是一个优先面向 Windows 的本地英语词汇学习桌面软件。它的核心目标是帮助英语考试和日常词汇学习者完成新学、复习、强化、测试、统计和备份流程，而不是扩展成在线教育平台。

本规格用于判断新功能是否属于 2.0 目标范围，避免功能无限扩张。

## 目标用户

- 准备 CET-4、CET-6、考研英语、IELTS、TOEFL 等考试的学习者。
- 需要日常积累英语词汇、希望离线使用的学习者。
- 希望学习进度、设置、收藏和备份保存在本机的用户。

## 支持平台

- 优先平台：Windows 10/11，依赖系统 WebView2。
- 源码运行：在 pywebview 支持的前提下，可兼容 macOS/Linux。
- 2.0 发布目标：Windows 可执行程序和基本发布包。

## 核心流程

1. 新学：选择词库后学习未掌握的新词。
2. 复习：根据 SM-2 间隔重复数据复习到期单词。
3. 强化：集中练习错误次数较多、EF 较低或已收藏的薄弱词。
4. 测试：用选择题检查词义掌握情况。
5. 统计：查看学习数量、连续学习、正确率和近期趋势。
6. 备份：导出、恢复和重置本地学习数据。

## 2.0 范围内功能

- 本地词库浏览和学习。
- 本地学习进度、设置、统计和收藏。
- 自定义单词导入/导出。
- ECDICT 考试标签词库 schema 校验。
- Windows 打包原型。
- 基础单元测试和关键 E2E 流程测试。

## 非目标功能

- 云同步。
- 登录、账号、会员或支付系统。
- 联网词库。
- AI 内容生成、AI 例句或 AI 学习计划。
- 宣称内置词库是完整官方考试大纲。
- 多人协作、班级管理或教师后台。

## 词库边界

当前内置词库由 ECDICT 的 `cet4`、`cet6`、`ky`、`ielts`、`toefl` 标签生成。它们覆盖 ECDICT 当前标注的对应考试词条，但不是教育部、考试院、IELTS 或 ETS 发布的官方考试大纲原件。任何发布材料都不能把它们宣传为“官方完整考纲”。

未来若更换或新增外部数据，必须先完成来源、许可、下载日期、版本/commit、字段映射脚本和去重规则记录，并通过 `scripts/validate_wordbanks.py` 校验。

## 版本规则

根目录 `VERSION` 是版本号单一来源。发布前必须确认以下文件中的版本一致：

- `VERSION`
- `package.json`
- `app.py`
- `README.md`
- `CHANGELOG.md`

版本一致性由 `npm run check` 中的 `scripts/check_version_consistency.py` 校验。

## 2.0 验收标准

- `npm test` 通过。
- `npm run check` 通过。
- `npm run e2e` 在 Windows 环境通过。
- 词库来源和许可状态记录在 `docs/wordbank-sources.md`。
- 内置词库通过 schema、重复词、空字段、超长字段和 metadata 一致性校验。
- 发布材料明确说明当前词库是 ECDICT 标签词库，不是官方考试大纲原件。
- Windows 打包产物能启动，且用户数据默认写入系统用户数据目录。
