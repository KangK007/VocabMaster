# VocabMaster Design System v3.0

## "Warm Dictionary" — 温暖词典美学

> **设计理念**：将古典词典的温度与现代瑞士排版的精确性融合，打造一个既专业又亲切的学习环境。

---

## 1. 设计方向

### 核心美学
VocabMaster v3.0 采用 **"Warm Dictionary"** 设计语言，灵感来源于：

- **古典词典**：温暖的纸质感、深色墨水文字、考究的排版层次
- **瑞士现代主义**：克制的色彩系统、精确的间距比例、清晰的视觉层次
- **学习空间**：专注、舒适、鼓励持续使用的氛围

### 设计差异点
| 维度 | v2.x (旧版) | v3.0 (新版) |
|------|------------|------------|
| 色彩基调 | 青绿色系 (#2f6f73) | 温暖纸白 + 墨水蓝 + 琥珀金 |
| 单词展示字体 | 系统无衬线 | Georgia 衬线 (更有词典感) |
| 数据字体 | 系统字体 | JetBrains Mono (等宽数据) |
| 侧边栏 | 72px 宽 | 60px 紧凑 |
| 卡片质感 | 通用白色卡片 | 纸质感微纹理 |
| 阴影系统 | 通用投影 | 纸质分层阴影 |
| 暗色模式 | 深灰调 | 暖黑调 + 降低对比 |

## 3. 排版系统

### 字体堆栈

| 用途 | 字体 | 理由 |
|------|------|------|
| 单词展示 | Georgia, "Noto Serif SC", serif | 词典感、优雅、可读 |
| UI 界面 | -apple-system, "PingFang SC", sans-serif | 清晰、原生性能 |
| 数据/代码 | "Cascadia Code", SFMono-Regular, monospace | 等宽对齐 |

### 字号比例 (模块化比例 1.25)

```
10px  — 标签、辅助信息
11px  — 小号 UI 文字
12px  — 标准 UI 文字
13px  — 按钮、强调 UI
14px  — 正文
17px  — 中型标题
20px  — 页面标题
clamp(28px, calc(var(--font-size) * 2), 44px) — 单词展示 (基于用户设置的流体尺寸)
```

### 用户可调字体

用户可在设置中调整 `--font-size` (14px–28px)，所有相关元素按比例缩放。

## 6. 动效系统

### 缓动函数

```css
--ease-out-expo:  cubic-bezier(0.16, 1, 0.3, 1)    /* 精确停止 */
--ease-out-quart: cubic-bezier(0.25, 1, 0.5, 1)     /* 平滑停止 */
```

### 持续时间

```
--duration-fast:   150ms   (微交互：hover、active)
--duration-normal: 250ms   (过渡：slide、fade)
--duration-slow:   400ms   (入场动画)
```

### 动效原则

- 只动画 `transform` 和 `opacity`（GPU 加速）
- 使用指数缓出创造"精确到达"的感觉
- 禁用动画时尊重 `prefers-reduced-motion`
- 卡片切换使用 180ms 滑入/滑出动画

## 8. 组件架构

### 8.1 导航组件

**侧边导航项 (.side-nav-item)**
- 默认：透明 + 灰色文字
- 悬停：浅色背景 + 边框显现
- 激活：墨水蓝浅底 + 蓝色文字
- 触觉反馈：点击缩小至 96%

**分类/模式标签 (.cat-tab / .mode-tab)**
- 分段控制器模式
- 激活态：墨水蓝填充 + 白色文字
- 混合模式：墨水蓝到琥珀金的渐变

### 8.2 单词卡片 (.word-card)

核心交互组件，承载所有学习体验：

- 纸质感微纹理 (径向渐变模拟)
- 悬停：边框琥珀色显现 + 阴影加深
- 滑动切换动画 (180ms 指数缓出)
- 包含子区域：单词展示、学习计划、进度指示

### 8.3 操作按钮

**评级按钮 (.rating-action)**
- 三个状态变体：认识(success绿) / 模糊(ink蓝) / 忘记(warning橙)
- 最小触摸目标：50px 高 × 104px 宽
- 快捷键提示在右上角
- 下次复习预览在标签下方

**确认按钮 (.btn-confirm)**
- 墨水蓝填充 + 投影
- 悬停加深至 #1f2650

### 8.4 弹窗系统

- 背景模糊 (`backdrop-filter: blur(6px)`)
- 内容卡片带边框 + 投影
- 滑入动画 (12px 向上 + 淡入)
- 设置弹窗：左侧导航 + 右侧表单的双栏布局
