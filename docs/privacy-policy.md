# VocabMaster 隐私说明

更新日期：2026-07-16

## 数据处理范围

VocabMaster 是离线优先的桌面软件，不提供账号、云同步、广告或遥测功能。软件本身不收集、上传或出售个人信息。

学习进度、统计、设置、收藏、自定义词库和本地安全备份默认保存在当前 Windows 用户的 `%APPDATA%\VocabMaster` 目录。导出文件只会写入用户在系统文件对话框中选择的位置。

## 系统能力

- 发音使用系统 WebView2 提供的语音合成能力。具体语音是否需要联网由 Windows、WebView2 和用户安装的语音包决定，VocabMaster 不控制该行为。
- 学习提醒使用 Windows 本地通知能力。
- 安装器检测到缺少 WebView2 Runtime 时，可由用户选择打开 Microsoft 的下载页面；不会自动下载或静默安装该组件。

## 数据保留与删除

用户可在软件中导出、恢复或重置学习数据。卸载程序默认保留 `%APPDATA%\VocabMaster`，以便升级或重新安装后继续使用。

如需彻底删除数据，请先退出 VocabMaster，再删除 `%APPDATA%\VocabMaster`。删除前建议先导出备份，此操作不可撤销。

## 发布前待补充

软件市场提交前，发布者必须在此处补充真实、可联系的支持地址或隐私联系渠道。本项目当前未获得该信息，因此不提供占位邮箱或虚构网址。
