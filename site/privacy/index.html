隐私说明

VocabMaster 是本地运行的桌面软件。以下内容用于说明软件如何处理学习数据，以及用户如何联系发布者。

发布者与隐私联系

发布者：邵睿康。

隐私联系页面：https://KangK007.github.io/VocabMaster/privacy/

数据存储

软件默认将学习进度、统计、设置、收藏、自定义词库和本地备份保存在当前系统用户的数据目录。Windows 默认位置为 %APPDATA%\VocabMaster。

联网行为

当前版本的核心学习功能可离线使用。若系统缺少 WebView2 Runtime，安装器可能提示用户打开 Microsoft 官方下载页面。

导出与备份

用户可以主动导出学习档案。导出的 JSON 文件未加密，用户应自行妥善保存，避免把包含学习记录的文件发送给不可信对象。

卸载与删除

卸载程序默认不删除用户学习数据。若需要清除全部学习数据，请在卸载后手动删除 %APPDATA%\VocabMaster。

隐私联系

一般问题可通过 GitHub Issues 联系发布者。请不要在公开 Issue 中提交个人联系方式、备份文件、学习档案或其他敏感信息。
