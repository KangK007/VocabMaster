常见问题

  • 无法打开窗口：请确认 Windows 10/11 已安装 Microsoft Edge WebView2 Runtime。
  • 安装后找不到数据：学习进度默认保存在当前 Windows 用户的数据目录，不写入安装目录。
  • 需要迁移数据：在应用设置中导出学习档案，再在新电脑中恢复备份。
  • 需要彻底删除数据：卸载软件后手动删除 %APPDATA%\VocabMaster。

问题反馈需要包含

  • 软件版本，例如 2.0.0。
  • Windows 版本和是否安装 WebView2 Runtime。
  • 问题发生时的操作步骤。
  • 是否使用过导入、恢复备份或开机自启。
